Proactive Learning

While traditional systems require the user to take an action to view content, GlowStone is proactive and automatically loads the exhibit's profile once the user is nearby, making it easier for the user to access content, and learn.

User Experience Focused

By using the power of the Estimote Beacon, we're able to provide superior user experience, as compared to traditional headset, QR code, or exhibit number methods. All the user needs to do is to install the app and walk near an exhibit.

Easy Configuration

We've provided a clean and intuitive web interface for you to register beacons. Only four blanks are required - the Beacon's ID, the exhibit name, a description, and a link to an image. After that, all that needs to be done is to register the beacon and place it near an exhibit.

Contextually-Aware

GlowStone can sense landmarks and attractions relevant to you using Estimote beacons, providing a totally seamless experience.

Information of Things

GlowStone uses beacon technology to provide contextual information about any location you want.
